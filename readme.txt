# Инструкция для администратора
Администратор имеет доступ к следующим функциям:
*Назначить специалиста* — позволяет назначить специалиста на конкретную заявку.
*Просмотреть заявки* — отображает все текущие заявки.
*Просмотреть специалистов* — отображает список всех специалистов, доступных для работы.
*Статистика* — отображает общее количество заявок и средний рейтинг специалистов.

# Инструкция для пользователей
Пользователи могут выполнять следующие действия:
*Создать заявку* — создайте новую заявку, выбрав категорию и описав проблему.
*Проверить статус заявки* — узнайте текущий статус своих заявок.
*Оставить отзыв* — оставьте отзыв по завершенной заявке.
*Регистрация* — зарегистрируйтесь в системе, указав свои данные.

# Инструкция для специалистов
Специалисты могут выполнять следующие действия:
*Мои заявки* — просмотрите заявки, назначенные для вас.
*Отправить отчет* — отправьте отчет о выполненной работе.
Главное меню*— вернитесь в главное меню.
Для начала работы с ботом используйте команду /start и выберите нужный пункт в меню.

#База данных
Бот использует базу данных SQLite (`bot_database.db`), в которой хранятся следующие таблицы:
-users— информация о пользователях.
-requests— информация о заявках.
-specialists— информация о специалистах.
-feedback— отзывы пользователей.
-reports— отчеты специалистов.

# Дополнительные настройки
-WAL-режим— включен для повышения производительности базы данных.
-Конфиденциальность — пользователи должны дать согласие на обработку персональных данных при регистрации.
При первом запуске бота база данных инициализируется автоматически.

# Зависимости
Для корректной работы бота используются следующие библиотеки:
Python 3.7 или выше
Библиотеки: telebot ,sqlite3
`Telebot` — для взаимодействия с Telegram API.
`sqlite3` — для работы с базой данных SQLite.
